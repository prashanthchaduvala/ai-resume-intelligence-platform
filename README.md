# AI Resume Intelligence Platform

An AI-powered platform that analyzes resumes against job descriptions and provides:

- Match Score
- Missing Skills Analysis
- Resume Improvement Suggestions
- AI-generated Interview Questions
- AI-generated Cover Letters

## Architecture

Frontend
- React

Backend Microservices
- Django (Auth Service)
- FastAPI (Resume Service)
- FastAPI (Analysis Service)
- FastAPI (AI Service)
- FastAPI (Notification Service)

Infrastructure
- Docker
- Kubernetes
- Terraform
- GitHub Actions

Observability
- Prometheus
- Grafana

Data Layer
- PostgreSQL
- Redis
